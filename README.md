# 🌩️ Cloud Vault

A secure, scalable, and modern cloud-based file management system with fullstack implementation.

## 🔧 Tech Stack

### Backend:
- Node.js
- Express.js
- MongoDB
- JWT, bcrypt
- Docker

### Frontend:
- React / Next.js
- Tailwind CSS
- Axios
- TypeScript (optional)
